GitHub 文件加速

GitHub文件链接带不带协议头都可以，支持release、archive以及文件，右键复制出来的链接都是符合标准的。

注意，不支持项目文件夹

分支源码：https://github.com/lers/project/archive/master.zip

release源码：https://github.com/lers/project/archive/v0.1.0.tar.gz

release文件：https://github.com/lers/project/releases/download/v0.1.0/example.zip

分支文件：https://github.com/lers/project/blob/master/filename
